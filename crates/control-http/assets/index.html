                Fonctions du node

                Une fonction désactivée est réellement arrêtée (socket fermée, port libéré, pipeline vidéo relâché) : elle ne consomme plus rien. Réactivée, elle redémarre immédiatement, sans relancer le node. Les choix sont mémorisés (fonctions.json) et survivent au redémarrage.

                Santé des fonctions

                Chargement…
                Pastille verte = fonction active (onglet Fonctions pour basculer). Les tuiles ci-dessus couvrent la machine ; erreurs et dérive de synchro y figurent aussi.

                Diagnostic

                Exporter le diagnostic (ZIP)
                Une archive avec l'état complet, le journal, les infos système, les écrans, les médias et les presets — à joindre quand on demande de l'aide. Aucun secret dedans.

                Mise à jour (expérimental)

                Trois temps prudents : vérifier la dernière version publiée, télécharger le binaire à côté (rien n'est remplacé), puis appliquer — le node redémarre sur la nouvelle version. En cas de pépin à n'importe quelle étape, l'installation actuelle reste intacte.

                Chataigne

                Télécharger Chataigne Vérification…
                Chataigne (logiciel séparé, gratuit) pilote le node en OSC : grâce à OSCQuery (port 8081), il découvre tout seul lecture, volume, mapping, effets, cues et lumières. Le node le détecte s'il est installé sur cette machine et peut le lancer d'ici.

                Alimentation de la machine

                Pour les installations pilotées à distance. Le node redémarre avec la machine (démarrage automatique / systemd).
